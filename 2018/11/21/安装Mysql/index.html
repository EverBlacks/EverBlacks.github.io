安装Apache

安装httpd 和 mod_ssl

httpd是Apache超文本传输协议(HTTP)服务器的主程序。被设计为一个独立运行的后台进程，它会建立一个处理请求的子进程或线程的池。apache2 以上的版本改称为httpd
mod_ssl 是Apache使用数字证书的模块

启动Apache服务

使用nano 或者 vi修改Apache 的配置文件

在文件的最下方添加ServerName的配置项

然后重启Apache 服务

想要在外部访问该服务，还需要开启端口

测试服务器是否能正常访问， 在浏览器中输入http://your Ip Address， 如果能看到Apache CentOS 默认主页就说明已经成功的安装并启动了Apache 服务。

service mysqld start ## 启动
iptables -I INPUT -p tcp –dport 3306 -j ACCEPT ##开放3306 端口

service iptables restart ## 重启防火墙

CREATE USER ‘root‘@’%’ IDENTIFIED BY ‘root’; ## 创建新用户

安装TomCat

  1. 下载Tomcata
  2. 通过FileZilla 或者其它的FTP 工具上传到服务器 或者直接使用 wget 下载 wget tomcat-download-url
  3. 解压 tar -zxv -f apache-tomcat-8.5.5.tar.gz
  4. 启动Tomcat tomcatPath/bin/startup.sh
  5. 开启8080端口
  6. 验证 浏览器访问http://your IP Address:8080
